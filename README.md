# Calculator

A simple calculator built with Python and Tkinter.

# Overview 

This project is a desktop calculator application that performs basic mathematical operations through a clean and user-friendly interface.

The goal of this project was to practice Python programming, GUI development with Tkinter, and event handling.

# Features

- Decimal number support
- Clear (C) button
- Backspace button 
- Error handling for invalid operations
- Simple and intuitive user interface
- Real-time expression display

# Technologies used
- Python
- Tkinter

# How to Run

1. Make sure Python is installed on your system
2. Download or clone the project folder
3. Open terminal in the project folder
4. Run:
```bash 
python main.py
```
